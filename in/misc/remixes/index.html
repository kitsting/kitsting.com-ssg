Heaven Studio Remixes

Lately I've been really getting into making Rhythm Heaven custom remixes using a wonderful program called Heaven Studio. This page hosts all the finished remixes I've made so far.

If you want to play them, you'll need the latest Bobby Build of Heaven Studio.

And it goes without saying, if you haven't played a Rhythm Heaven Game before, you totally should! I'm not a rhythm game person in the slightest but I found myself really enjoying them. If you own a Nintendo 3DS or a Nintendo PC, they are fairly accessible to play with the right methods...

Feisty Flowers

Everhood OST - Cazok

This is my first remix! Made for the Grayfruit Rhythm Heaven contest, it ended up getting second place!

Honestly I think this one came out really good. I wasn't planning to participate in the contest, but had a vision of the Everhood-themed Fork Lifter and just had to do it. Making the custom graphics was a lot of fun!

I think the 1-week time limit mostly worked in my favor, since I had to lock in and focus on the essentials first. This led to me actually finishing it!

Power of Katamari Damacy

Once Upon a Katamari OST - Shigeru Matsuzaki (and others)

My second remix, I opted for a slightly slower song and I chose this one because I thought the "almond chocolate" line was funny. I don't think it came out as good as my first remix, but it still came out pretty good.

I forced myself to rush to finish this one after procrastinating on it for 2 weeks or so, so some parts could have been better.

I thought that lipsyncing the frog was gonna be a pain, but it wasn't too bad and the result is definitely worth it.
