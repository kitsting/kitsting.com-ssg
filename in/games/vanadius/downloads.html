Vanadius Version Archive

← Go back to the main game page


This is an archive of several versions of Vanadius for historical purposes. While it does not contain every version I have (there are way too many), I opted to include a good mix of versions that show how the game improved over time.

While you can download release versions from this page, I highly recommend downloading those from the itch.io page instead if you can. It saves on website bandwidth and you get access to itch.io's nice features!


Note: All the versions here are Windows builds (that work fine in Wine on Linux at the time of writing - 2025/8/24). Younger me didn't have the foresight to make native Linux builds.
